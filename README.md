## yml 세팅
api-key : 본인 ChatGPT API Key
model : 사용할 모델 (gpt-4, gpt3.5, gpt-4o 등등 모델에 대한 자세한 정보는 [여기](https://platform.openai.com/docs/models) 참조)
image.options.model = 추후 강의에서 제공될 이미지 생성 모델
