# 프롬포트 엔지니어링의 기초

while (원하는 응답이 나올때까지) {

1. AI에게 역할을 부여하기
2. AI가 해야할 일 정의하기
3. 예시 질문과 응답을 통하여 원하는 답변 이끌어내기
4. “네 알겠습니다.” 와 같은 맥락이 필요없다면 말하지 말라 지정하기

}

**프롬포트 엔지니어링의 고급언어**

제로샷 (Zero-shot) : 예제를 하나도 안주고 답변을 유도하는 기술

퓨샷 (Few-shot) : 예제를 몇가지 주고 답변을 유도하는 기술
